# vscode

Unified VS Code config source for both Linux and macOS.

## Linux

- Settings path: `.config/Code/User/settings.json`
- The canonical tracked settings file now lives at `home/.config/Code/User/settings.json`.

## macOS

- Settings path: `Library/Application Support/Code/User/settings.json`
- `home-manager` links this path from `home/.config/Code/User/settings.json`.

## Shared

- `extensions.txt` is the shared extension list.
